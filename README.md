I built a FastAPI backend for file uploads. This project is for learning purpose only. There is nothing too advanced.


Create .env file and paste your cred. :
DATABASE_URL=?
AWS_ACCESS_KEY_ID=?
AWS_SECRET_ACCESS_KEY=?
AWS_REGION=?
AWS_S3_BUCKET=?


Tech Stack:
- FastAPI
- PostgreSQL
- AWS S3
- JWT Authentication
